# Carpeta de imágenes — Spax

Esta carpeta está organizada **por sección**, igual que el orden de la página. Cada subcarpeta trae una imagen de marcador de posición (con la paleta de Spax) lista para que reemplaces por la foto final — solo respeta el **mismo nombre de archivo y la misma proporción** para que el diseño no se descuadre.

| Carpeta | Dónde se usa | Tamaño recomendado |
|---|---|---|
| `hero/` | Imagen principal del Hero (mujer / retrato) | 900×1000 px (vertical) |
| `about/` | Banda de info + sección "Nosotros" (imagen principal y flotante) | 1600×380 / 540×360 px |
| `servicios/` | 3 tarjetas de servicios | 520×620 px c/u |
| `porque-elegirnos/` | Sección "Por qué elegirnos" | 700×760 px |
| `proceso/` | Video / procedimiento | 760×640 px |
| `como-trabajamos/` | Sección "Cómo trabajamos" | 640×720 px |
| `beneficios/` | Retrato circular en banda oscura | 520×620 px |
| `antes-despues/` | 2 comparativas antes/después | 640×480 px c/u |
| `testimonios/` | Avatares de pacientes | 120×120 px |
| `equipo/` | 4 fotos de especialistas | 380×460 px c/u |
| `blog/` | 3 miniaturas de artículos | 460×320 px c/u |
| `contacto/` | (reservada, el mapa usa Google Maps embebido) | — |

**Recomendación:** usa fotografías con licencia propia o de bancos con licencia comercial (no Pinterest ni capturas de redes sociales), en formato `.jpg` o `.webp` optimizado para web (idealmente bajo 300 KB por imagen).
